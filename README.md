# KIS Market Dashboard

한국투자증권 API 기반 개인 투자 대시보드.

## 목표
- 시장 주요 지수와 상태를 한눈에 확인
- 관심 종목 가격/등락률/거래대금 모니터링
- 추후 Vercel 배포 및 개인화 대시보드 확장

## Planned Features
- KOSPI / KOSDAQ / 환율 / 주요 지표 카드
- 관심 종목 Watchlist
- 시장 breadth / movers / sector overview
- Vercel 배포

## Stack (planned)
- Next.js
- TypeScript
- Tailwind CSS
- Korean Investment & Securities API
